# Better Auth POC

A monorepo project demonstrating better authentication practices.

## Project Structure

```
packages/
  ├── backend/    # Backend API service
  └── frontend/   # Frontend application
```

## Getting Started

### Prerequisites

- Node.js >= 18.0.0
- pnpm >= 8.0.0

### Installation

```bash
pnpm install
```

### Development

```bash
# Start all services
pnpm dev

# Build all packages
pnpm build

# Run tests
pnpm test

# Run linting
pnpm lint
```
